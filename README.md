# Student Performance Analysis

This project analyzes student academic data to understand how study hours and attendance affect exam performance.

## Tools Used
- Python
- pandas
- matplotlib
- Google Colab

## Project Features
- Data exploration and cleaning
- Performance analysis
- Data visualization
- Insight generation

## Key Insights
- Students who study more than 5 hours score higher
- Attendance above 80% improves performance

Open the notebook file in this repository.
